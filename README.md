# 🎮 Hangman Game – Python Mini Project

A simple and interactive text-based Hangman Game developed using Python.

This project was created as part of an internship task to demonstrate fundamental programming concepts such as:
- Loops
- Conditions
- Lists
- Strings
- Randomization

---

# 📌 Project Overview

The Hangman Game is a classic word-guessing game where the player attempts to guess a hidden word one letter at a time.

## The player:
- Wins by correctly guessing all letters in the word
- Loses if the number of incorrect guesses reaches the maximum limit

This project is completely console-based and designed for beginners to understand core Python concepts in a practical way.

# 📖 Game Rules

- The player must guess the hidden word.
- Only one letter can be entered at a time.
- Maximum incorrect attempts allowed: **6**
- Repeated guesses are not allowed.

The game ends when:
- The player guesses the word correctly
- OR the player runs out of attempts
